# Форкни это!

1. Создайте аккаунт на [Github](https://github.com).

2. Зайдите на [репозиторий]() и справо-сверху нажмите на кнопку Fork.

3. Назовите вашу репозиторию (обычно название хака).

4. Можете прочитать эту [инструкцию на английском](https://docs.github.com/en/github/working-with-github-pages/creating-a-github-pages-site) для подключения github pages к вашему репозиторию.

5. Если вы хотите сделать локальные изменения в вашей репозитории то обратитесь к [Инструкции по подаче материалов](/submissions/submission-guide), там есть разные советы по редактированию и созданию материалов.

## Создание PDF
1. Установите любое на выбор приложение для создание файлов формата pdf, обычно это Adobe InDesign или Affinity by Canva.
2. Рекомендуем сортировать заклинания и предметы по алфавиту.
3. Запишите в титры всех, кто работал над pdf (Авторы текста, корректоры, редакторы, художники и т.д.).
